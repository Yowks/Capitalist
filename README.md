# Converter
 
 MDS Capitalist project with nw.js

# Installation
Use `git clone` to install this app.

```
git clone https://github.com/Yowks/Capitalist.git
```

Then use : 

```
npm i
```

# Run

To run the program you need to run :

```
npm run dev
```

And to build the app for differents os who will be generated in the dist repository: 

```
npm run prod
```
